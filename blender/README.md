# Introduction
This folder contains the 3D printed parts.  I printed all of the parts in ASA for longevity but any material will do.

<img src="../pictures_for_readme/3d_printed_parts.png" alt="Blender" style="width:50%; height:auto;">

# Printing Instructions
All parts should be printed at fastest speed, 15% in-fill.  No supports should be required aside from the lid which will need supports on the build plate for the screw hole.

You have a choice where the wheel is concerned: print either `wheel_for_unmodified_handle` or `wheel_for_handle_no_sheath`.  I found that the sheath on the handle, designed to make it easier to operate by hand, wrattled irritatingly and so I took it off and printed `wheel_for_handle_no_sheath` into which I fitted a 4&nbsp;x&nbsp;8&nbsp;x&nbsp;3&nbsp;mm bearing.  If you prefer not to take the handle sheath off, print `wheel_for_unmodified_handle` instead.

# Introduction
This folder contains the 3D printed parts.  I printed all of the parts in ASA for longevity but any material will do.

<img src="../pictures_for_readme/3d_printed_parts.png" alt="Blender" style="width:50%; height:auto;">

# Printing Instructions
All parts should be printed at fastest speed, 15% in-fill, probably in black, aside from `base_outer` which you may wish to print in natural ASA, a better colour match for the plywood that it will be sandwiched between.  No supports should be required for any part except for the lid, which will need supports on the build plate for the screw hole.

You have a choice where the wheel is concerned: print either `wheel_for_unmodified_handle` or `wheel_for_handle_no_sheath`.  I found that the sheath on the handle of the plinky-plonky mechanism, designed to make operating it by hand easier, wrattled irritatingly and so I took the sheath off and printed `wheel_for_handle_no_sheath` into which I fitted a 4&nbsp;x&nbsp;8&nbsp;x&nbsp;3&nbsp;mm bearing to take the inner part of the handle.  If you prefer not to remove the sheath, print `wheel_for_unmodified_handle` instead.
